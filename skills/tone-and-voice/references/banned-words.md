# Banned Words & Phrases

Words and phrases that conflict with Maplerun's tone. Each entry includes a suggested alternative.

## Corporate Jargon

| Avoid | Use Instead |
|-------|-------------|
| Leverage (verb) | Use |
| Utilize | Use |
| Facilitate | Help |
| Synergy / Synergistic | Work together / Connected |
| Paradigm shift | Big change |
| Best-in-class | Best |
| Cutting-edge | Newest / Most advanced |
| Robust | Strong / Reliable |
| Scalable | Grows with you |
| Seamless | Smooth / Easy |
| Streamline | Simplify |
| Unlock | Get / Access |
| Empower | Help / Let |
| Holistic | Whole / Complete |
| Deep dive | Look closely at |

## Hype & Clickbait

| Avoid | Use Instead |
|-------|-------------|
| Game-changer | Significant improvement |
| Disruptive / Disruption | New approach |
| Revolutionary | New |
| Groundbreaking | First |
| World-class | Excellent |
| Next-level | Better |
| Move the needle | Make progress |
| Circle back | Follow up |
| Touch base | Check in |
| Low-hanging fruit | Quick win |

## Filler & Padding

| Avoid | Reason |
|-------|--------|
| "Hope this finds you well" | Impersonal filler |
| "Please don't hesitate to..." | Weak and wordy |
| "At your earliest convenience" | Vague; just say when |
| "As per my last email..." | Passive-aggressive |
| "Going forward" | Redundant |
| "It goes without saying" | Then don't say it |
| "To be honest with you" | Implies you're not usually honest |
| "With all due respect" | Disrespectful signal |

## Urgency Language (Spam Triggers)

Never use these in email subject lines:

- FREE (all caps)
- URGENT
- Act now
- Limited time (unless genuinely true — and even then, be specific)
- Don't miss out
- Click here
- You've been selected
